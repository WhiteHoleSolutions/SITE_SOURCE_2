# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-06

### Added
- 🎉 Initial release
- ✨ Public website with hero banner and portfolio
- 🔐 Customer portal with private galleries
- 👤 Admin panel with comprehensive management tools
- 📄 Invoice system with PDF generation
- 💰 Bill of sale tracking
- 📊 Expense tracking with GST calculation
- 📈 Analytics dashboard with financial reports
- 🔍 Inquiry management with status tracking
- 🎨 Fully responsive design with Tailwind CSS
- 🗄️ SQLite database with Prisma ORM
- 🚀 Render.com deployment configuration
- 📱 Mobile-first UI
- 🔒 JWT authentication with HTTP-only cookies
- 🎯 Role-based access control
- 📦 Image upload and management
- 🔐 Secure password hashing
- 📝 Business settings configuration
- 💳 Revolut payment integration (optional)
- 📧 Customer password reset functionality
- 📝 Admin notes on customer profiles

### Security
- Implemented JWT authentication
- Added password hashing with bcryptjs
- Protected API routes with middleware
- Implemented role-based access control
- Added CSRF protection
- Input validation and sanitization
